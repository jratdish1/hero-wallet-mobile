# Deployment Checklist

## Pre-Deployment

- [ ] Chain ID verified.
- [ ] Contract addresses verified.
- [ ] Constructor and initializer arguments reviewed.
- [ ] Owner/admin/multisig addresses reviewed.
- [ ] Timelock/governance delay reviewed.
- [ ] Oracle addresses and decimals reviewed.
- [ ] Deployment account has only required permissions.
- [ ] Dry run completed.
- [ ] Tests and security checks passed.

## Deployment

- [ ] No private keys printed.
- [ ] No secrets committed.
- [ ] Broadcast output archived securely.
- [ ] Verification metadata captured.
- [ ] Events and monitoring configured.

## Post-Deployment

- [ ] Contract verified on explorer.
- [ ] Ownership transferred where applicable.
- [ ] Implementation contracts locked.
- [ ] Monitoring alerts active.
- [ ] Runbook updated.
